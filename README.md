### Gtk.AlertDialogについて

#### Gtk.AlertDialogのプロパティ

| プロパティ | 内容 |
| --- | --- |
| buttons | アラートに表示されるボタンのラベル。 |
| cancel-button | Escキーが押されたときに何が起こるかを決定します。 |
| default-button | Returnキーが押されたときに何が起こるかを決定します。 |
| detail | アラートの詳細テキスト。 |
| message | アラートのメッセージ。 |
| modal | アラートがモーダルかどうか。 |
